# VolunAI Flutter App

## Setup

1. Make sure Flutter is installed: https://flutter.dev/docs/get-started/install
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to launch the app

## Structure
- lib/screens/ - All app screens
- lib/widgets/ - Reusable widgets
- lib/theme/ - Colors and theming
- assets/images/ - App assets

## Features
- Login / Sign Up
- Role selection (Volunteer / Organizer)
- Home feed with events
- Search by category
- Event details with join button
- User profile with analytics
- AI Chat (VolunAI)
- VolunGo map game
- Leaderboard/Rating
- Notifications
- Psychology AI consultation
- AI-powered post generator
