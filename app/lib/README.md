# VolunAI Flutter App



## Structure
- lib/screens/ - All app screens
- lib/widgets/ - Reusable widgets
- lib/theme/ - Colors and theming
- assets/images/ - App assets

## Features
- Login / Sign Up
- Role selection (Volunteer / Organizer)
- Home feed with events
- Search by category
- Event details with join button
- User profile with analytics
- AI Chat (VolunAI)
- VolunGo map game
- Leaderboard/Rating
- Notifications
- Psychology AI consultation
- AI-powered post generator
